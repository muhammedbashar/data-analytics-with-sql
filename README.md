# 🧠 SQL Data Analytics Portfolio

## 📋 Project Overview
This repository showcases a complete **SQL Data Analytics project** designed and implemented using **MySQL Workbench**.  
It demonstrates real-world data analytics workflows — from database setup and data exploration to performance reporting and KPI insights — entirely using SQL.

---

## 🎯 Objectives
- Build and manage a **mini data warehouse** in MySQL.  
- Perform **exploratory and analytical SQL operations** on sales, customer, and product datasets.  
- Generate **reports and insights** to support business decision-making.  
- Apply **advanced SQL functions** to calculate key performance metrics.

---

## ⚙️ Tools & Technologies
- **Database:** MySQL 8.0  
- **Interface:** MySQL Workbench  
- **Data Format:** CSV  
- **Key Concepts:** Joins, CTEs, Window Functions, Subqueries, Case Logic, Views, Aggregations
